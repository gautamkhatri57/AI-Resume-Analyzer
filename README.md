# AI Resume Analyzer

## Overview

AI Resume Analyzer is a Python-based application that analyzes resumes and identifies relevant skills from a predefined dataset.

## Features

* Resume Upload
* Skill Extraction
* Resume Analysis
* Easy-to-use Interface

## Technologies Used

* Python
* Streamlit
* Pandas

## Project Structure

* app.py
* skills.csv
